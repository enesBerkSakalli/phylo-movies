# PhyloMovies README

## Summary

`README.md` is the public project overview for [[phylo-movies]]. It positions
PhyloMovies as a browser-based and desktop-capable phylogenetic tree viewer that
animates topological differences between consecutive trees as interpretable
subtree migrations. Source: `README.md`.

## Key Claims

- PhyloMovies targets sliding-window phylogenetic analyses of MSAs and bootstrap
  replicate inspection. Source: `README.md`.
- The tool complements Robinson-Foulds and other summary distances by showing
  which lineages move, where they move from, and what new groupings form.
  Source: `README.md`.
- The frontend is JavaScript/React and the backend is [[brancharchitect]], a
  Python engine included as a git submodule. Source: `README.md`.
- The GitHub Pages deployment is documentation-only and does not run backend
  workflows. Source: `README.md`.
- FastTree is positioned as a rapid exploratory bundled workflow, while
  publication-grade users can load externally inferred trees from tools such as
  IQ-TREE or RAxML-NG. Source: `README.md`.
