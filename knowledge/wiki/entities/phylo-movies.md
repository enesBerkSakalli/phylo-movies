# PhyloMovies

## Summary

PhyloMovies is a browser-based and desktop-capable phylogenetic tree viewer for
animating topology changes between ordered trees. Source: `README.md`.

## Key Claims

- It targets sliding-window MSA analyses, recombination exploration, and
  bootstrap/rogue-taxon inspection. Source: `README.md`.
- It combines a React/deck.gl frontend with the [[brancharchitect]] backend for
  SPR path computation and transition-frame generation. Source: `README.md`.
- It is distributed under the MIT License. Source: `README.md`.
- Its public project page is documentation-only; full workflows need desktop,
  Docker, or local full-stack execution. Source: `README.md`.
