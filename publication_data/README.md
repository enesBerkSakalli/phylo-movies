# Phylo-Movies Publication Data

This folder contains data used to demonstrate and validate Phylo-Movies in the
manuscript. It intentionally separates third-party source material from
Phylo-Movies-derived inputs, outputs, and scripts.

Start here:

- `PUBLICATION_DATA_RELATIONSHIP_MAP.md` explains which external paper or
  repository each dataset comes from and which files Phylo-Movies actually uses.
- `norovirus/README.md` maps the recombination example.
- `norovirus/RECOMBINATION_DATA_HYGIENE_AUDIT.md` records the current ReCAN
  hygiene blockers.
- `bootstrap_example/README.md` maps the Aberer/RogueNaRok-derived bootstrap
  examples.
- `rogue_taxa/README.md` maps the rogue-taxon scripts and external-data policy.

## Current Top-Level Groups

| Group | Purpose | Ownership |
| --- | --- | --- |
| `norovirus/` | Norovirus recombination example and ReCAN validation workspace. | Derived from Nextstrain/GenBank source data; Phylo-Movies outputs are ours. |
| `bootstrap_example/` | Phylo-Movies-ready rogue-taxon bootstrap examples for datasets 24 and 125. | Derived from selected Aberer et al. alignment inputs; generated Phylo-Movies demo outputs are ours. |
| `rogue_taxa/` | Rogue-taxon publication scripts, provenance notes, and selected external-file policy. | Scripts/provenance are ours; full Aberer/RogueNaRok archive is external and not vendored. |
| `figure_example/` | Small figure/demo tree input. | Phylo-Movies manuscript support file. |
| `quick_msa_demo/` | Synthetic lightweight MSA demo. | Synthetic Phylo-Movies demo data. |

## Reader Rule

If a file is external source material, it should be described with source,
citation, access date, and transformation status. If a file is generated by
Phylo-Movies or by our helper scripts, it should point back to the source inputs
and command or run log that produced it.
