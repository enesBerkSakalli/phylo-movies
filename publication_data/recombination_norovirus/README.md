# Norovirus Recombination Publication Layer

This folder is the publication-facing home for norovirus recombination source
inputs.

## Contents

| Path | Role |
| --- | --- |
| `source_alignments/` | Source and derived FASTA inputs used for the norovirus Phylo-Movies and ReCAN workflows. |
| `source_alignments/MANIFEST.tsv` | Source, checksum, sequence count, site count, role, and notes for each source or derived alignment. |
| `REGENERATE.md` | Reviewer-facing commands for regenerating the ReCAN validation results from this source layer. |
