# Phylo-Movies Manual

Phylo-Movies helps researchers inspect how phylogenetic trees change across ordered analyses such as sliding-window inference, bootstrap replicate review, and curated tree-series comparisons.

Use this manual when you want to:

- load example or local tree data,
- understand the visualization workspace,
- interpret the movie timeline and transition inspector,
- connect tree changes with MSA windows when alignment data are available,
- export figures or WebM recordings,
- troubleshoot backend and browser-demo behavior.

The GitHub Pages browser demo contains precomputed examples. Uploading new data and running inference require the desktop app, Docker workflow, or local BranchArchitect backend.
