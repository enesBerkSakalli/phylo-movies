# Feature Reference

This section explains the controls and methods available after a dataset is loaded or prepared.

Use it when you need to answer practical questions such as:

- which setup settings affect tree inference,
- how branch-length and layout controls change the tree view,
- how to interpret the movie timeline and Transition Inspector,
- how MSA windows are synchronized with tree frames,
- how taxa coloring, subtree highlighting, and SPR analytics work,
- what the PNG and WebM export buttons save.

Most controls are context-sensitive. If a dataset does not include an MSA, branch annotations, generated transition frames, or SPR metadata, the matching controls are hidden, disabled, or report unavailable values.
